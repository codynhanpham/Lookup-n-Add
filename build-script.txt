Run these two commands in the same directory as the package.json file:

npx esbuild index.js --bundle --platform=node --target=node14 --outfile=build_node14.js
npx pkg build_node14.js -t node14 -o dist/lookup-n-add


If build for multiple platforms:
npx pkg build_node14.js -t node14-linux,node14-win,node14-macos -o dist/lookup-n-add
